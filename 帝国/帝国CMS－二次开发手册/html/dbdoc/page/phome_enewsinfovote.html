phome_enewsinfovote 数据表字段解释（信息投票表）

字段名 类型 解释 附加说明
id int(11) 信息ID
classid smallint(6) 栏目ID
title varchar(120) 投票标题
voteip mediumtext 投票IP记录
votetext text 投票项目 格式：(每个项目用回车隔开)
项目名称1::::::投票数1
项目名称2::::::投票数2
voteclass tinyint(1) 投票类型 0为单选，1为多选
doip tinyint(1) 是否限制同一IP投票 0为不限制，1为限制
dotime date 过期时间
tempid smallint(6) 投票模板ID
width int(11) 查看投票窗口宽度
height int(11) 查看投票窗口高度
diyotherlink tinyint(1) 是否手动相关链接 1为手动相关链接，0为查询相关链接
infouptime int(10) 信息上线时间 unix时间戳
infodowntime int(10) 信息下线时间 unix时间戳
